# Phase V Edge Targets

Phase V searches for places where Circle Calculus might provide leverage that is hard, awkward, or unavailable in ordinary presentations. The target is not metaphor. The target is a concrete artifact: a proof interface, benchmark, search index, provenance object, explanatory workflow, or Living Book path.

The source of truth is `manifests/phase5_edge_targets.yaml`, validated by `scripts/check_phase5_targets.py` and exported to `site/data/generated/phase5_targets.json`.

## Discipline

Every Phase V target must name an ordinary baseline, acceptance criteria, artifact refs, dictionary dependencies, a next action, and a guardrail. Planned targets are not proof claims. Experimental targets are not Lean proofs. A direction should be deferred if it cannot produce a concrete artifact beyond a metaphor.
